# Braintree Laravel



## Requirements

- PHP v8.4
- Laravel v12
- Web Server (HTTPS)

## Setup

- `composer create-project laravel/laravel braintree-laravel`
- `cd braintree-laravel`

## Deploy

SFTP to your favourite hosting provider, make sure it's HTTPS.
